# EduGen

EduGen is an open-source educational toolkit designed to help teachers create high-quality learning materials quickly.

## Features

- Generate lesson notes
- Create objective and theory examinations
- Produce marking schemes
- Generate worksheets
- Support Nigerian curriculum
- Export educational content

## Why EduGen?

Teachers spend hours preparing instructional materials. EduGen automates repetitive tasks while allowing teachers to customize the final output.
